# 🛒 Simba Supermarket

Simba is an online supermarket built for Rwandan shoppers — making it easy to discover, compare, and buy everyday groceries and household goods from the comfort of home. Whether you're stocking up on Inyange milk, Akabanga chilli oil, or fresh produce from the local market, Simba puts Kigali's shelves in your pocket.

By the end of this course we will have shipped six core features: **browse** a full product catalogue organised by category, **search** for any item by name or keyword, **filter** results by price, category, or availability, manage a **cart** that persists across your session, complete a **checkout** flow with order summary and payment details, and **deploy** the app live to the web so real shoppers can use it.
